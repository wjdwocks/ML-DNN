### 기본적인거
- 연합학습이란?
    - 원시 데이터를 중앙 서버로 모으지 않고, 여러 클라이언트(기기/기관)가 로컬에서 모델을 학습한 뒤 모델 업데이트(가중치 변화) 만을 중앙에서 집계해 글로벌 모델을 만드는 분산 학습 방식임
- 연합학습의 기본 동작 흐름.
    1. 서버가 초기 모델 파라미터 w0 배포.
    2. 각 클라이언트 i는 자신의 로컬 데이터로 모델을 E epoch/batch 만큼 학습해 로컬 업데이트 wi를 얻음.
    3. 클라이언트는 서버에 로컬 업데이트(그라디언트)를 압축하거나 요약하여 전송.
    4. 서버는 수신된 업데이트들을 집계하여 새로운 글로벌 파라미터 wt+1 = $\sigma ni/N wi$ 하여 계산함.
    5. 업데이트된 글로벌 모델을 다시 클라이언트들에게 배포하고, 이를 반복함.
    - 여기에서 중요한거는 서버는 데이터가 아니라, 모델 업데이트만 받기 때문에, 애초에 데이터를 옮김으로써 생길 수 있는 보안 위협이 없음..
    - 하지만 데이터 포이즈닝과 같이, 로컬 데이터 자체가 악의적으로 조작되어 학습되면, 그 클라이언트의 업데이트가 글로벌 모델을 망가뜨리거나 백도어를 심는다.
